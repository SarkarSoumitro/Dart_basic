   ########## Important notice ##########


1. If a variable is private its can not be access by different class. (Private variable)

2. In dart if i use ("_") before a variable , method , class it will considered as the private .

3. when we don't want to use direct instance then we use abstract class .

4. "late"  keywords is use before declare a variable it means that this variable is not nullable , later it will be initialized.

5. In generics synchronous and asynchronous operation are held.

6. In steam we use this keywords => async* , yield (instead of return) , in the main several ways to receive value from the steam function.

7. By using mixim a class can inherit multiple class  by "with" keywords.
